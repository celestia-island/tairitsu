# エンタープライズサポート

Tairitsuは、本番アプリケーションを展開する組織向けにエンタープライズ級サポートを提供しています。

## サポートレベル

### コミュニティサポート（無料）

- 公開ドキュメントとガイド
- コミュニティDiscordとGitHubディスカッション
- 公開リリースによるバグ修正
- ベストプラクティスドキュメント

### プロフェッショナルサポート

専用サポートを必要とする組織は以下をご利用ください：

- 優先的なバグ修正とセキュリティパッチ
- コア開発者への直接アクセス
- カスタム統合サポート
- アーキテクチャレビューとガイダンス
- SLA対応のレスポンスタイム

### エンタープライズサポート

エンタープライズライセンスには以下が含まれます：

- 24/365緊急問題サポート
- 専任サポートエンジニア
- オンサイトトレーニングとワークショップ
- カスタム機能開発
- 長期サポート（LTS）バージョン
- セキュリティ監査支援

## サポートの利用

### コミュニティチャンネル

- **GitHub Issues**: バグ報告と機能リクエスト
- **GitHub Discussions**: Q&Aとディスカッション
- **Discord**: リアルタイムコミュニティチャット

### プロフェッショナル向けお問い合わせ

メール：enterprise@tairitsu.dev

以下を含めてください：
- 組織名と規模
- プロジェクトの説明と使用例
- 具体的なサポート要件
- 期待されるタイムライン

## SLA（サービス品質保証）

### プロフェッショナルサポート

| 重大度 | レスポンス時間 | 解決時間 |
|--------|----------------|----------|
| 緊急 (P1) | 4時間 | 2営業日 |
| 高 (P2) | 1営業日 | 5営業日 |
| 中 (P3) | 2営業日 | 2週間 |
| 低 (P4) | 1週間 | ベストエフォート |

### エンタープライズサポート

| 重大度 | レスポンス時間 | 解決時間 |
|--------|----------------|----------|
| 緊急 (P1) | 1時間 | 24時間 |
| 高 (P2) | 4時間 | 2営業日 |
| 中 (P3) | 1営業日 | 1週間 |
| 低 (P4) | 2営業日 | 2週間 |

## 重大度の定義

### P1 - 緊急
- 本番システムが完全に停止
- データの損失または破損
- アクティブに悪用されている脆弱性

### P2 - 高
- 主要な機能が動作しない
- 重大なパフォーマンス低下
- 回避策はあるが生産性に影響

### P3 - 中
- 軽微な機能の問題
- 不便な回避策が必要
- ドキュメントのエラー

### P4 - 低
- 外観の問題
- 機能リクエスト
- 一般的な質問

## 長期サポート（LTS）

LTSバージョンは以下を提供：

- 2年間のバグ修正
- 3年間のセキュリティ更新
- 重要な問題のバックポート
- 安定したAPI保証

現在のLTSバージョン：なし（初のLTS：v1.0.0）
