# Tairitsu - SVG & Build Resources Plan

## Overview

Simplify icon support into two focused components:
1. **tairitsu core**: Safe SVG dispatch (replaces dangerouslyInnerHTML)
2. **tairitsu-packager**: build.rs resource indexing

## Motivation

- SVG inline embedding is common and should be safe by default
- `dangerouslyInnerHTML` is a security risk - need a safe alternative
- Resources (SCSS, SVG) should be auto-discovered at build time
- Eliminate manual resource path configuration

## Design

### Part 1: Safe SVG in tairitsu-core

Add to `packages/vdom/src/`:

```rust
/// Safe SVG content with XSS protection
pub struct SafeSvg {
    content: String,
}

impl SafeSvg {
    /// Create safe SVG from validated content
    /// - Removes script tags
    /// - Removes event handlers (onclick, onload, etc.)
    /// - Removes external references
    pub fn new(content: &str) -> Self;

    /// Create from compile-time validated constant
    pub const fn from_static(content: &'static str) -> Self;
}

impl VElement {
    /// Set safe SVG content (sanitized)
    pub fn safe_svg(self, svg: SafeSvg) -> Self;
}
```

### Part 2: build.rs Resource Indexing in tairitsu-packager

Add `packages/packager/src/build.rs` support:

```rust
/// Resource index generated by build.rs
pub struct ResourceIndex {
    /// SCSS files with their output paths
    pub scss: Vec<ResourceEntry>,
    /// SVG files with their content
    pub svg: Vec<ResourceEntry>,
}

pub struct ResourceEntry {
    pub source_path: PathBuf,
    pub output_path: PathBuf,
    pub hash: String,  // Content hash for cache busting
}
```

### Part 3: Macro Integration

```rust
// In rsx! macro:
rsx! {
    svg {
        // Safe SVG from indexed resource
        safe_svg: svg!("icons/sun.svg"),
    }
}

// SCSS integration:
scss!("styles/main.scss")  // Auto-indexed by build.rs
```

## API

```rust
// Safe SVG usage
let icon = SafeSvg::from_static(r#"<path d="M12 2L2 22h20L12 2z"/>"#);

rsx! {
    svg {
        viewBox: "0 0 24 24",
        safe_svg: icon,
    }
}

// Build-time resource access
let styles = scss!("main.scss");  // Auto-resolved via build.rs
let icon = svg!("icons/sun.svg"); // Auto-resolved via build.rs
```

## Security

SVG sanitization removes:
- `<script>` tags
- Event handlers (onclick, onload, onerror, etc.)
- External references (xlink:href to external URLs)
- JavaScript URLs (javascript:...)
- Data URLs with executable content
